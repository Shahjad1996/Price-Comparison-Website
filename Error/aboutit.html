Overview of our Work

BEST 'O' MANIA explores the possibility of creating a user friendly mobile-price comparison web application that portrays latest mobile prices from top e-commerce websites like Amazon, Flipkart, Snapdeal, etc. This application also serves as a ideal tool for businesses that need to see exactly what their competitors are offering, including product types. At the same time, this application provides the quickest way to analyze prices and find the right products at the right price.
